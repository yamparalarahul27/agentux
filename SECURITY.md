# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AgentUX, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, email **rahulvignanwork@gmail.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix and release**: As soon as possible, depending on severity

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Scope

This policy covers the `@yamparala27/agentux` npm package and the source code in this repository.
